# Spypa v1.0 Beta

## Cara Install

1. **Clone repo ini:**

```bash
git clone https://github.com/demonhunters999/Spypa
cd Spypa
```

2. **Install dependensi:**

```bash
pkg update -y
pkg install nodejs git zip -y
```

4. **Jalankan bot untuk pertama kali:**

```bash
node spypa.js
```
